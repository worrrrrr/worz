## 🎯 ภาพรวม

WORZ SOVEREIGN CORE v2.1 เป็นระบบ Neuro-Symbolic AI Agent ที่สามารถประมวลผลคำสั่งซับซ้อนได้หลายรูปแบบ:
- **คณิตศาสตร์** (Math Engine)
- **ตรรกะ** (Logic Engine)
- **แปลงหน่วย** (Unit Engine)
- **ทั่วไป** (General Engine)
- **ฐานความรู้** (Knowledge Base - 69 สูตร/กฎ)

## 📋 โครงสร้างคำสั่งพื้นฐาน

```
[คำบอกลำดับ] [คำสั่งหลัก] [รายละเอียด] [คำเชื่อม] [คำสั่งรอง] [ระดับความสำคัญ]
```

### องค์ประกอบของคำสั่ง

| องค์ประกอบ | ตัวอย่างภาษาไทย | ตัวอย่างภาษาอังกฤษ | ความหมาย |
|-----------|----------------|-------------------|----------|
| **คำบอกลำดับ** | ขั้นแรก, ก่อนอื่น, ต่อไป, สุดท้าย | first, next, then, finally | กำหนดลำดับการดำเนินการ |
| **คำสั่งหลัก** | คำนวณ, หา, แปลง, ตรวจสอบ | calculate, find, convert, check | ประเภทการประมวลผล |
| **คำเชื่อม** | และ, แต่, หรือ, แล้ว | and, but, or, then | เชื่อมหลายคำสั่ง |
| **คำปฏิเสธ** | ไม่ต้อง, อย่า, ห้าม, ไม่ | no, not, don't, never | ยกเลิกหรือปฏิเสธ |
| **เงื่อนไข** | ถ้า, หาก, เมื่อ | if, when, unless | กำหนดเงื่อนไข |
| **ความสำคัญ** | ด่วน, สำคัญ, ปกติ, ต่ำ | urgent, important, normal, low | ลำดับความสำคัญ |

## 🔢 1. คำสั่งคณิตศาสตร์ (Math Engine)

### คำสั่งพื้นฐาน
```
"คำนวณ [นิพจน์คณิตศาสตร์]"
"หา [ผลลัพธ์] ของ [สมการ]"
```

### ตัวอย่าง

#### ง่ายๆ
```
✅ "คำนวณ 2 บวก 2"
✅ "หา 5 คูณ 3"
✅ "10 หาร 2"
✅ "sqrt(144)"
✅ "2^10"
```

#### สมการ
```
✅ "แก้สมการ x + 5 = 12"
✅ "หา x จาก 2x = 10"
✅ "x^2 - 4 = 0"
```

#### หลายนิพจน์ (ใช้คำเชื่อม)
```
✅ "คำนวณ 2 บวก 2 และ 3 คูณ 4"
→ ผลลัพธ์: [4, 12]

✅ "หา 10 หาร 2 แต่ 5 บวก 5"
→ ผลลัพธ์: [5.0, 10]
```

#### ใช้สูตรจาก Knowledge Base
```
✅ "คำนวณพื้นที่วงกลมที่มีรัศมี 10 เมตร"
→ ดึงสูตร: area_circle = π * r²
→ ผลลัพธ์: 314.159

✅ "หาเส้นรอบวงของวงกลมรัศมี 5"
→ ดึงสูตร: circumference = 2 * π * r
→ ผลลัพธ์: 31.4159

✅ "คำนวณพื้นที่สามเหลี่ยมที่มีฐาน 10 สูง 5"
→ ดึงสูตร: area_triangle = 0.5 * b * h
→ ผลลัพธ์: 25.0
```

#### ฟิสิกส์ (ดึงสูตรจาก KB)
```
✅ "หาแรงที่ใช้เคลื่อนมวล 50 กิโลกรัมด้วยความเร่ง 2 m/s²"
→ ดึงสูตร: F = m * a
→ ผลลัพธ์: 100 นิวตัน

✅ "คำนวณพลังงานจลน์ของมวล 10 kg ความเร็ว 5 m/s"
→ ดึงสูตร: KE = 0.5 * m * v²
→ ผลลัพธ์: 125 จูล

✅ "หาความเร็วมุมลงของวัตถุตกอิสระจากความสูง 20 เมตร"
→ ดึงสูตร: v = sqrt(2 * g * h)
→ ผลลัพธ์: 19.8 m/s
```

## 🧠 2. คำสั่งตรรกะ (Logic Engine)

### คำสั่งพื้นฐาน
```
"[ประพจน์] เป็นจริงหรือไม่"
"ตรวจสอบว่า [เงื่อนไข]"
```

### ตัวอย่าง

#### ตรรกะพื้นฐาน
```
✅ "true และ false"
→ ผลลัพธ์: False

✅ "จริง หรือ เท็จ"
→ ผลลัพธ์: True

✅ "ไม่(true)"
→ ผลลัพธ์: False
```

#### เงื่อนไข (If-Then)
```
✅ "ถ้าฝนตก แล้วถนนเปียก"
→ ประเมินความสัมพันธ์เชิงเหตุผล

✅ "หาก x > 5 แล้ว x เป็นจำนวนบวก"
→ ตรวจสอบเงื่อนไข
```

#### หลายขั้นตอน
```
✅ "ขั้นแรก ตรวจสอบว่า A เป็นจริง ขั้นต่อไป ตรวจสอบว่า B เป็นเท็จ สุดท้าย สรุปผล"
→ แสดงผลทีละขั้นตอน
```

## 📏 3. คำสั่งแปลงหน่วย (Unit Engine)

### คำสั่งพื้นฐาน
```
"แปลง [ค่า] [หน่วยเดิม] เป็น [หน่วยใหม่]"
"[ค่า] [หน่วยเดิม] เท่ากับกี่ [หน่วยใหม่]"
```

### หน่วยที่รองรับ

| ประเภท | หน่วยไทย | หน่วยอังกฤษ |
|--------|---------|-----------|
| **ความยาว** | เมตร, กิโลเมตร, เซนติเมตร, มิลลิเมตร | m, km, cm, mm, ft, in |
| **น้ำหนัก** | กิโลกรัม, กรัม, ปอนด์, ออนซ์ | kg, g, lb, oz |
| **อุณหภูมิ** | องศาเซลเซียส, องศาฟาเรนไฮต์, เคลวิน | c, f, k |
| **ปริมาตร** | ลิตร, มิลลิลิตร, แกลลอน | l, ml, gal |
| **เวลา** | วินาที, นาที, ชั่วโมง | s, min, h |

### ตัวอย่าง

#### เดี่ยว
```
✅ "แปลง 50 องศาเซลเซียสเป็นฟาเรนไฮต์"
→ ผลลัพธ์: 122°F

✅ "10 กิโลกรัมเป็นปอนด์"
→ ผลลัพธ์: 22.0462 lb

✅ "100 เมตรเป็นฟุต"
→ ผลลัพธ์: 328.084 ft

✅ "1 ชั่วโมงเป็นวินาที"
→ ผลลัพธ์: 3600 s
```

#### หลายการแปลง (Multi-step)
```
✅ "ขั้นแรก แปลง 100 องศาฟาเรนไฮต์เป็นเซลเซียส ต่อไป แปลง 50 กิโลกรัมเป็นปอนด์"
→ แสดงผลทีละขั้นตอนพร้อมลำดับ
```

## 🎛️ 5. คำสั่งควบคุมลำดับขั้น (Sequence Control)

### คำบอกลำดับ

| ภาษาไทย | อังกฤษ | ความหมาย |
|--------|--------|----------|
| ขั้นแรก / ก่อนอื่น | first / firstly | ขั้นตอนที่ 1 |
| ขั้นที่สอง / ต่อไป | second / next | ขั้นตอนที่ 2 |
| ขั้นที่สาม | third | ขั้นตอนที่ 3 |
| สุดท้าย / ขั้นสุดท้าย | finally / last | ขั้นตอนสุดท้าย |

### ตัวอย่างคำสั่งซับซ้อน

#### แบบ 2 ขั้นตอน
```
✅ "ขั้นแรก คำนวณพื้นที่วงกลมรัศมี 5 เมตร ต่อไป แปลง 100 องศาฟาเรนไฮต์เป็นเซลเซียส"
→ Step 1: พื้นที่วงกลม = 78.54 m²
→ Step 2: 100°F = 37.78°C
```

#### แบบ 3 ขั้นตอน
```
✅ "ก่อนอื่น หาแรงจากมวล 10 kg ความเร่ง 3 m/s² ต่อไป คำนวณพลังงานจลน์จากมวลเดียวกันความเร็ว 5 m/s สุดท้าย แปลงผลลัพทธ์เป็นหน่วยอื่น"
→ Step 1: F = 30 N
→ Step 2: KE = 125 J
→ Step 3: (แปลงหน่วยตามที่ต้องการ)
```

## 🧩 9. คำสั่งผสมผสาน (Multi-Engine Complex Commands)

### ตัวอย่างระดับสูง

#### แบบที่ 1: Math + Unit + Sequence
```
✅ "ขั้นแรก คำนวณพื้นที่วงกลมรัศมี 10 เมตร ต่อไป แปลงผลลัพธ์เป็นตารางฟุต สุดท้าย แสดงผลด่วน"
→ Step 1: 314.159 m²
→ Step 2: 3381.96 ft²
→ Priority: urgent
```

#### แบบที่ 2: Physics + Math + Logic
```
✅ "ถ้ามวล 50 kg และความเร่ง 2 m/s² ให้หาแรง แล้วตรวจสอบว่าแรงมากกว่า 100 N หรือไม่"
→ Step 1: F = 100 N
→ Step 2: Logic check → False (เท่ากับ 100 ไม่มากกว่า)
```

#### แบบที่ 3: Multi-step Physics Problem
```
✅ "ก่อนอื่น คำนวณความเร็วมุมลงของวัตถุตกจากความสูง 20 m ต่อไป หาพลังงานจลน์เมื่อกระทบพื้น ถ้ามวล 5 kg สุดท้าย แปลงพลังงานเป็น kJ"
→ Step 1: v = 19.8 m/s
→ Step 2: KE = 980 J
→ Step 3: 0.98 kJ
```

#### แบบที่ 4: Complex Engineering Calculation
```
✅ "ขั้นแรก หาแรงโน้มถ่วงระหว่างมวล 1000 kg กับ 500 kg ห่างออกไป 10 m ต่อไป คำนวณพลังงานศักย์โน้มถ่วง ถ้าต้องการยกมวล 1000 kg สูง 5 m สุดท้าย เปรียบเทียบทั้งสองค่า"
→ Step 1: F = G * (m1*m2) / r²
→ Step 2: PE = m * g * h
→ Step 3: Comparison result
```
